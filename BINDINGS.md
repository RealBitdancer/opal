# opal bindings and wrappers

Bindings expose opal to languages other than C. The known ones are listed here. If you
wrote one, or know of one that is missing, send a pull request and it will be added.

## Language bindings

| Name                                                  | opal version | Language                   | License |
| :---------------------------------------------------- | :----------: | :------------------------: | :-----: |
| [opal-zig](https://github.com/RealBitdancer/opal-zig) | 1.0.2        | [Zig](https://ziglang.org) | MIT     |

Bindings usually follow the naming convention `opal-{language}`.
